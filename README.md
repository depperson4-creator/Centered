## Run locally

## Deploy (Vercel recommended)
- Push to a GitHub repo or use Vercel's "Deploy from your computer"
- Build: `npm run build`
- Set your domain, then update `metadataBase` in `app/layout.tsx` and `public/robots.txt` + `public/sitemap.xml`

## Google Analytics
